CloudTestPackagePYPI
==============

Installation
--------------

CloudTestPackagePYPI can be installed by unzipping the source code in one directory and using this command: ::

    sudo python setup.py install

You can also install it directly from the Python Package Index with this command: ::

    sudo pip cloudtestpackagepypi install
